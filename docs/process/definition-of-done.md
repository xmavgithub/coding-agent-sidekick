# Definition of Done

A feature is done only if:
- Research and plan exist and are up to date.
- All annotation cycle notes are addressed.
- Planned tests were executed successfully.
- No open blockers remain.
- PR is complete with artifact references.
- For `existing` mode: compatibility, regression, and rollback controls are documented and validated.
- For `existing` mode: baseline and context pack are updated after code changes.
