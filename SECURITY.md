# Security Policy

If you find a vulnerability, do not open a public issue with sensitive details.

## Reporting
- Send a private report to project maintainers.
- Include:
  - Problem description
  - Estimated impact
  - Reproduction steps
  - Possible fix or mitigation

## Target SLA
- Initial acknowledgement: within 3 business days.
- Assessment and remediation plan: within 7 business days.
